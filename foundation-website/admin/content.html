Razorpay Integration

Razorpay dashboard (https://dashboard.razorpay.com) से Key ID और Key Secret लेकर यहाँ paste करें। Donate page पर "Pay Online" button activate हो जाएगा।

Setup Guide:

  1. https://dashboard.razorpay.com पर जाएं
  2. Sign up / Login करें
  3. Settings → API Keys → Generate Key
  4. Key ID copy करके ऊपर paste करें
  5. "Save" करें — Done! Website पर payment button active हो जाएगा
